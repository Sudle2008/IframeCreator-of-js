# IframeCreator-of-js

提供函数：
CreateIframe(url,h)
| 参数名 | url | h |
| :-: | :-: | :-: |
| 默认值 | 无 | 窗口高度 |
| 类型 | str | str |
| 解释 | 所添加网页地址 | 网页高度 |

默认返回值：1

类型：int

默认控制台输出："succeed to add Iframe"
